Slider valueNotify() Demonstration.

  1. Select a slider from the radio buttons at the top of the screen.
  2. You will see which sliders have the valueNotify() turned on.
  3. Sliders with the valueNotify() turned on will react when they move past the tolerance value.
  4. The ranges in use are: #1 ( 0 to 100), #2 (10 to 600), #3 (0 to 100)
  5. Click any of the buttons to see the selected slider moved to that position.
